Panduan Bermula

  2. Tambah Senarai Murid: Masukkan semua nama murid dalam kelas anda satu per satu.
  3. Pilih Tab "Kemasukan Borang": Mulai merekod penilaian harian murid mengikut komponen.
  4. Lihat Analisis: Tab "Ringkasan Laporan" menunjukkan statistik dan pencapaian murid secara keseluruhan.
  5. Cetak Laporan Rasmi: Gunakan tab "Transkrip Murid" untuk menghasilkan sijil dan laporan individu.
  6. Simpan & Backup: Semua data disimpan di peranti anda. Saran: Eksport data secara berkala untuk backup.

Sambungan & Konfigurasi Sistem

Sila lengkapkan pautan URL Google Web App anda serta Tahun Sesi Akademik semasa di bawah. Tetapan ini akan disimpan secara kekal di dalam memori pelayar peranti harian anda.

Panduan Pemutus & Penyegerakan Google Sheets

Bagi memastikan fungsi pemadaman dipautkan ke Google Sheet pada ruangan baris yang betul (*backward loop*), sila pastikan kod Web App di bawah disalin sepenuhnya ke **Extensions > Apps Script** di Google Sheet anda, kemudian deploy semula sebagai Web App baharu.

Kod Apps Script Code.gs: 
function doPost(e) {
  var lock = LockService.getScriptLock();
  lock.tryLock(10000);
  try {
    var requestData = JSON.parse(e.postData.contents);
    var action = requestData.action;
    var ss = SpreadsheetApp.getActiveSpreadsheet();
    var sheet = ss.getSheets()[0]; 
    
    if (sheet.getLastRow() === 0) {
      sheet.appendRow([
        'ID', 'Nama', 'Bidang', 'Komponen', 'Tarikh',
        'TP', 'Eviden', 'Pautan Fail Bukti'
      ]);
    }
    
    if (action === 'delete') {
      var id = requestData.id;
      var values = sheet.getDataRange().getValues();
      for (var i = values.length - 1; i >= 1; i--) {
        if (values[i][0] == id) { 
          sheet.deleteRow(i + 1);
        }
      }
      return ContentService.createTextOutput(JSON.stringify({ success: true })).setMimeType(ContentService.MimeType.JSON);
    }
    
    var fileUrl = "";
    if (requestData.fileData && requestData.fileName) {
      try {
        var folder, folders = DriveApp.getFoldersByName("Evidens Pentaksiran Prasekolah");
        if (folders.hasNext()) {
          folder = folders.next();
        } else {
          folder = DriveApp.createFolder("Evidens Pentaksiran Prasekolah");
        }
        var decoded = Utilities.base64Decode(requestData.fileData);
        var blob = Utilities.newBlob(decoded, requestData.fileType, requestData.fileName);
        var file = folder.createFile(blob);
        file.setSharing(DriveApp.Access.ANYONE_WITH_LINK, DriveApp.Permission.VIEW);
        fileUrl = file.getUrl();
      } catch (fErr) {
        fileUrl = "Ralat muat naik Drive: " + fErr.toString();
      }
    } else {
      fileUrl = requestData.fileName || "";
    }
    
    sheet.appendRow([
      requestData.id, requestData.nama, requestData.bidang,
      requestData.komponen, requestData.tarikh, requestData.tp,
      requestData.eviden, fileUrl
    ]);
    return ContentService.createTextOutput(JSON.stringify({ success: true, fileUrl: fileUrl })).setMimeType(ContentService.MimeType.JSON);
    
  } catch (error) {
    return ContentService.createTextOutput(JSON.stringify({ success: false, error: error.toString() })).setMimeType(ContentService.MimeType.JSON);
  } finally {
    lock.releaseLock();
  }
}
